    JSON 내보내기 · GitHub 배포

    1단계. "파일 다운로드" 버튼을 누르면 quotes.json 파일이 내려받아집니다. 현재 브라우저에 저장된 모든 고객 견적이 담겨 있습니다.

    2단계. GitHub 저장소(예: trademark)에 이 파일을 quotes.json 이름으로 업로드합니다. 기존 파일이 있으면 덮어쓰기 하세요.

    3단계. 1~2분 뒤 https://내이름.github.io/trademark/ 에 접속한 모든 고객이 업데이트된 견적을 볼 수 있습니다.

    ※ 고객명·암호·견적 정보가 공개 저장소에 올라가므로 민감한 정보는 담지 마세요. 암호는 간단한 접근 제한용이며 보안용이 아닙니다.
